    • O que são tags?

    As tags HTML são os elementos básicos usados para definir a estrutura e o conteúdo de uma página da web. Cada tag HTML é composta por uma palavra-chave cercada por colchetes angulares (< e >). As tags geralmente são usadas em pares, com uma tag de abertura e uma tag de fechamento, separadas pelo conteúdo que elas envolvem.

    Tag <html>

    Indica o início do documento HTML.

    Tag <head>

    Contém informações sobre o documento, como o título e os links para arquivos externos.

    Tag <body>

    Contém o conteúdo principal do documento, como textos, imagens e outros elementos.

    Tag <h1> ao <h6>

    São usadas para criar cabeçalhos em uma página HTML. O número após o "h" na tag indica o nível de importância do cabeçalho, sendo que o <h1> é o mais importante e o <h6> é o menos importante. O objetivo dessas tags é organizar visualmente o conteúdo da página e indicar ao usuário qual é a estrutura de tópicos da página.

    Tag <p>

    É usada para definir um parágrafo em uma página HTML. Ela é uma das tags mais comuns em HTML, e é usada para separar o texto em blocos de parágrafos. Quando você envolve um pedaço de texto com a tag <p>, o navegador irá automaticamente adicionar uma quebra de linha antes e depois do texto, criando um espaço em branco entre os parágrafos.

    Tag <a>

    É usada para criar links em uma página HTML. Ela tem um atributo obrigatório chamado href, que especifica o URL de destino do link. Além disso, a tag <a> pode ter outros atributos opcionais, como target (que especifica onde o link deve ser aberto - na mesma janela ou em uma nova janela ou aba do navegador), rel (que define a relação entre a página atual e a página de destino do link, como "nofollow", que indica aos motores de busca que o link não deve influenciar a classificação do site na pesquisa) e title (que fornece uma dica de ferramenta ao usuário quando o cursor do mouse é colocado sobre o link).

    Tag <img>

    É usada para inserir imagens em uma página HTML. Ela tem um atributo obrigatório chamado src, que especifica o URL da imagem a ser exibida. Além disso, a tag <img> pode ter outros atributos opcionais, como alt (que fornece uma descrição alternativa da imagem para usuários com deficiência visual ou para quando a imagem não pode ser exibida) e width e height (que especificam a largura e a altura da imagem em pixels).

    Tag <mark>

    É usada para destacar uma parte do texto que tem importância ou relevância especial em relação ao restante do conteúdo. Geralmente, o texto destacado é exibido com um fundo amarelo claro por padrão nos navegadores, embora isso possa variar dependendo do estilo definido no CSS.

    Tag <p>

    É usada em HTML para indicar que o texto dentro dela é de tamanho reduzido em relação ao restante do conteúdo da página. Por padrão, o texto dentro da tag <small> é exibido com uma fonte menor do que o texto normal.

    Essa tag pode ser útil para indicar informações secundárias ou menos importantes em um documento, como a data de publicação de um artigo, ou para indicar que o conteúdo de uma nota de rodapé é menor do que o texto principal.

    Tag <ul>/<ol> e <li>

    As tags <ul> e <ol> são usadas para criar listas em HTML, enquanto a tag <li> é usada para definir cada item da lista.

    A tag <ul> é usada para criar uma lista não ordenada, em que os itens são marcados com um ponto ou um círculo. Já a tag <ol> é usada para criar uma lista ordenada, em que os itens são numerados ou marcados com letras.

    A tag <li> é usada para definir cada item da lista, que pode ser um parágrafo, uma imagem, um link ou qualquer outro elemento HTML.

    Tag <div>

    As tags <div> e <span> são elementos de bloco e de linha, respectivamente, e são utilizadas para agrupar outros elementos HTML e aplicar estilos ou comportamentos específicos a eles.
